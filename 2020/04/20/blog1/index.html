html简介

  • HyperText Mark-up Language 超文本标记语言
  • 超文本包括文字，音频，视频，图片，动画等
  • 标记就是一个个的标签
  • 用于创建网页，HTML文档也叫做 web 页面

html笔记

<!DOCTYPE html>   <!--声明html5-->    
<html lang="en">  <!--lang标签 语言--> 
<head>            <!--head标签 头部-->
<meta charset="UTF-8">     <!--meta标签 元数据（描述性标签，描述网站信息如关键词，文件的最后修改时间，作者，和其他元数据）-->
<meta name="Keywords" content="关键字" />
<meta name="Description" content="简介、描述" />

<title>网页标题</title>       <!--title标签 网页标题-->
<link type="text/css" rel="stylesheet" href="**.css"/>  <!--link标签 链接css-->
<style type="text/css">       
嵌⼊入css样式代码
</style>
<script >
JavaScript脚本程序
</script>
</head>
<body>
<!-- 文本标签 -->    
<h3>网页显示内容</h3> <!--h1-h6 标题标签--> 
<i>i: 斜体标签</i> <br/>
<em>em: 强调斜体标签</em> <br/>
<b>b: 加粗标签</b><br/><br/>
<strong>strong: 强调加粗标签</strong><br/>
<del>del: 删除线</del><br/>
<u>u: 下划线</u> <br/><br/>
<!-- 格式化标签 -->   
<p>i: 段落标签</p> <br/>    <!--br 换行标签 --> <!--hr 水平线标签 --> 
列列表：
<ul>...</ul> 无序列列表
<ol>...</ol> 有序列列表 其中type类型值：A a I i 1 start属性表示起始值
<li>...</li> 列表项
<dl>...</dl> 自定义列列表
<dt>...</dt> 自定义列列表头
<dd>...</dd> 自定义列列表内容
<div>...</div> 常用于组合块级元素，以便便通过 CSS 来对这些元素进⾏行行格式化
<span>...</span> 常用于包含的⽂文本，您可以使⽤用 CSS 对它定义样式，或者使⽤用 JavaScript 对
它进行行操作。
<!-- 图像标签 -->   
 <img />
其中img标签中常⽤用属性如下：
src： 图⽚片名及url地址
alt: 图⽚片加载失败时的提示信息
title：⽂文字提示属性
width：图⽚片宽度
height：图⽚片⾼高度
border：边框线粗细
<!-- 超级链接标签a -->   
格式： <a href="链接⽬目标url地址">显示⽂文字</a>
a标签的属性：
href: 必须，指的是链接跳转地址
target: 表示链接的打开⽅方式：
_blank 新窗⼝口
_parent ⽗父窗⼝口
_self 本窗⼝口（默认）
_top 顶级窗⼝口
framename 窗⼝口名
title：⽂文字提示属性（详情）
锚点链接：
定义⼀一个锚点： <a id="a1"></a> 以前使⽤用的是 <a name="a1"></a>
使⽤用锚点： <a href="#a1">跳到a1处</a>
<!-- 表单标签form -->  
<form>
<fieldset>
<legend>个⼈人信息：</legend>
姓名：<input type="text" /><br/>
年年龄：<input type="text" /><br/>
</fieldset>
<br/><br/>
<fieldset>
<legend>健康信息：</legend>
身⾼高：<input type="text" /><br/>
体重：<input type="text" /><br/>
</fieldset>
</form>
城市：
<select name="city">
<optgroup label="河北北省">
<option>⽯石家庄</option>
<option>保定</option>
<option>廊坊</option>
</optgroup>
<optgroup label="河南省">
<option>郑州</option>
<option>安阳</option>
<option>周⼝口</option>
</optgroup>
</select>
<!-- 框架标签iframe -->  
属性：src：规定在 iframe 中显示的⽂文档的 URL
name：规定 iframe 的名称
height：规定 iframe 的⾼高度。
width：定义 iframe 的宽度。
frameborder：规定是否显示框架周围的边框。
例例如：<iframe src="1.html" name="myframe" width="700" height="500"></iframe>
</body>
</html>
